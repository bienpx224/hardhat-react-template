# hardhat-waffle-ethers-ts-template
Hardhat+Waffle+Ethers+Typescript template to compile and test a Solidity Smart Contract

Install:
```
npm install
```

Compile:
```
npm run compile
```

Test:
```
npm run test
```

Deploy to Rinkeby test network:
```
npm run deploy
```

Run prettier on all your contracts:
```
npm run prettier
```
